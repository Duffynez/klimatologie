# klimatologie

Tento repozitář obsahuje zdrojový kód pro webové stránky [klimatologie.eu](http://klimatologie.eu). Web je postavený na statickém generátoru stránek Jekyll a slouží jako informační portál o klimatologii v českém jazyce. Obsahuje články na různá témata, jako je globální průměrná povrchová teplota nebo ústup horských ledovců.
